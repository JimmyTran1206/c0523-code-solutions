# react-rendering-lists-quiz-notes

## Quiz Questions

After completing this exercise, you should be able to discuss or answer the following questions:

- When would we want to dynamically create a list of React components?

- to show a list of items

- Why is it important for React components to be data-driven?

- as data change, the UI changes, but the component behind stay the same and can be reusable.

- Where in the component code would a list of React components typically be built?

  - before the return statement, of course ha ha.

- What `Array` method is commonly used to create a list of React components?

- Array.prototype.map() and Array.prototype.filter()

- Why do components in a list need to have unique keys?

- so that react can access the data and update it when needed

- What is the best value to use as a "key" prop when rendering lists?

- ids, index, database keys, random keys...
