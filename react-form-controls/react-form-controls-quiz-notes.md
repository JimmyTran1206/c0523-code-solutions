# react-form-controls-quiz-notes

## Quiz Questions

After completing this exercise, you should be able to discuss or answer the following questions:

- How do controlled components differ from uncontrolled components?

- controlled components have state while uncontrolled component does not.

- What are some advantages of using uncontrolled components?

  - simpler code, takes all form entries at once

- What are some advantages of using controlled components?

- process each form control separately, good for customizing, error handling, and debugging.

- Which style do you prefer?

  - controlled form, seem more 'react'. Uncontrolled form seems more 'javascript'.

- What two props must you pass to an input for it to be "controlled"?

  -onChange, value

- What are some popular npm packages for creating forms in React?
- React Hook, Formik, React Final Form
